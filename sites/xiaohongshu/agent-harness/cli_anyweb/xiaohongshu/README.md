# cli-anyweb-xiaohongshu

This package wraps the shared `cli-anyweb` runtime with xiaohongshu-specific assets.

## Entry Point

- console command: `cli-anyweb-xiaohongshu`
- Python module: `cli_anyweb.xiaohongshu`

## Bundled Assets

- `skills/references/xiaohongshu.sitemap.md`
- `skills/evals/xiaohongshu.starter.eval.yaml`
- `skills/evals/xiaohongshu.starter.path.yaml`

## Site Commands

- `cli-anyweb-xiaohongshu home`
- `cli-anyweb-xiaohongshu search --query <query>`
- `cli-anyweb-xiaohongshu search-video --query <query>`
- `cli-anyweb-xiaohongshu search-video-open-first --query <query>`
- `cli-anyweb-xiaohongshu search-video-open-first-title --query <query>`
- `cli-anyweb-xiaohongshu search-video-open-first-by-html --query <query>`
- `cli-anyweb-xiaohongshu open-note --url <note-url>`
- `cli-anyweb-xiaohongshu open-note --note-id <id> --xsec-token <token>`
- `cli-anyweb-xiaohongshu open-first-note`
- `cli-anyweb-xiaohongshu note-detail --url <note-url>`
- `cli-anyweb-xiaohongshu note-detail --note-id <id> --xsec-token <token>`
- `cli-anyweb-xiaohongshu login`
- `cli-anyweb-xiaohongshu publish`
- `cli-anyweb-xiaohongshu notifications`
- `cli-anyweb-xiaohongshu profile`
